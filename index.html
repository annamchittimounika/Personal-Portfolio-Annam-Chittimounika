My Projects

Project Title:Secure Cloud-Based Healthcare Management System

Technologies Used:

Java, HTML, CSS, JSP, Servlets, JDBC, MySQL, NetBeans IDE, JFreeChart, Cloud Computing, Attribute-Based Encryption (ABE)

Description:

Developed a secure and scalable cloud-based healthcare management system that allows patients to register, upload medical reports, and receive doctor feedback online. The system integrates cloud storage with Attribute-Based Encryption (ABE) to ensure patient data privacy and prevent unauthorized access. JFreeChart is used to visualize patient health records and generate analytical reports, improving data interpretation for healthcare professionals.


Objective:

To design and implement a cloud-enabled healthcare platform that securely stores and manages medical data while providing easy access to patients and doctors with enhanced data privacy, reliability, and performance.



Guide:

Dr. D. Ravi Kiran, HOD of Computer Science Engineering Institution: Malineni Perumallu Educational Society's Group of Institutions (Affiliated to JNTU Kakinada)


Key Features:

  • Secure Cloud Storage: Stores patient data safely on a cloud platform with access control and encryption.
  • Attribute-Based Encryption (ABE): Ensures data privacy and allows only authorized users (patients/doctors) to access records.
  • Patient Registration Module: Enables patients to register and manage their personal and medical information online
  • Doctor Management Module: Allows doctors to review patient reports and provide medical suggestions digitally
  • Report Upload & Access: Patients can upload lab reports or prescriptions, and doctors can view them securely through the portal.
  • Data Visualization: Integrated JFreeChart to generate health analytics and visual charts for easy medical data interpretation.
  • User-Friendly Interface: Developed using JSP, HTML, and CSS for an intuitive and responsive user experience.
  • Database Integration: Uses MySQL with JDBC for efficient data storage, retrieval, and management.
  • Cloud-Based Architecture: Enables real-time data access from anywhere, improving healthcare service delivery.
  • Improved Reliability: Reduces manual record handling and data redundancy while enhancing accuracy and performance.

    • Outcome:

    Successfully achieved secure cloud storage and efficient data handling, reduced manual record maintenance, improved healthcare data accessibility, and ensured high-level data security using encryption and cloud integration.
